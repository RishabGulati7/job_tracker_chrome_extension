# ◆ Job Tracker

> Track job applications to Google Sheets with one click — powered by Claude AI.

## What it does

Extracts job details from any posting page (LinkedIn, company careers, job boards) and logs them to your Google Sheet in one click. Claude reads the page. You review. One click saves.

## Features
- AI extraction via Claude (Company, Role, Location, Salary, Team)
- Smart column mapping for existing sheets
- Duplicate detection
- HYPERLINK formulas linking back to the posting
- Editable review screen before saving
- Multi-tab Google Sheets support
- First-run onboarding wizard
